# Como Criar a Imagem Open Graph (og-image.png)

## ⚠️ IMPORTANTE - Ação Necessária

O site agora está configurado para usar `og-image.png`, mas você precisa criar essa imagem.

## Especificações da Imagem

- **Dimensões:** 1200x630 pixels (exato)
- **Formato:** PNG ou JPG
- **Nome do arquivo:** `og-image.png`
- **Local:** `site-valepan/assets/logo/og-image.png`

## Opções para Criar

### Opção 1: Ferramenta Online (Mais Fácil)

1. Acesse: https://www.canva.com/create/open-graph/
2. Crie um design 1200x630px
3. Use a logo da ValePan (valepan-logo-full.svg)
4. Adicione o texto: "ValePan - O Pão das Melhores Hamburguerias"
5. Subtexto: "Pães Premium desde 1991"
6. Exporte como PNG
7. Salve em `site-valepan/assets/logo/og-image.png`

### Opção 2: Photoshop/GIMP

1. Crie um novo arquivo 1200x630px
2. Importe a logo SVG
3. Adicione textos e elementos visuais
4. Exporte como PNG
5. Salve em `site-valepan/assets/logo/og-image.png`

### Opção 3: Usar Logo Existente (Temporário)

Se precisar de algo rápido:
1. Abra `valepan-logo-full.svg` em um navegador
2. Tire um screenshot
3. Redimensione para 1200x630px em um editor de imagens
4. Salve como `og-image.png`

## Recomendações de Design

- Use as cores da marca ValePan (dourado #C6A848 e vinho #3F0312)
- Inclua a logo centralizada ou à esquerda
- Adicione o slogan "Hambúrguer bom começa pelo pão"
- Mantenha textos grandes e legíveis
- Evite bordas muito próximas das extremidades

## Testar a Imagem

Após criar, teste em:
- Facebook Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- WhatsApp: Envie o link do site para alguém

## ✅ Depois que criar a imagem

Simplesmente coloque o arquivo em:
```
site-valepan/assets/logo/og-image.png
```

O site já está configurado para usar essa imagem automaticamente!
