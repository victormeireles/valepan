## ⚠️ ATENÇÃO - Última Ação Necessária

### Criar Imagem Open Graph

**O que é:** Imagem que aparece quando alguém compartilha seu site no WhatsApp, Facebook, etc.

**Como criar:** Veja instruções detalhadas em `CRIAR-OG-IMAGE.md`

**Especificações:**
- Tamanho: 1200x630 pixels
- Formato: PNG
- Local: `assets/logo/og-image.png`

**Sugestão rápida:**
1. Use Canva.com (gratuito)
2. Tamanho personalizado: 1200x630px
3. Adicione a logo ValePan
4. Texto: "ValePan - O Pão das Melhores Hamburguerias"
5. Exporte como PNG
6. Salve em `assets/logo/og-image.png`
